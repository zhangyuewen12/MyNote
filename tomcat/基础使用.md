### 测试启动

```

# 测试启动
cd /usr/local/soft/tomcat9/bin
./catalina.sh start
# ./catalina.sh stop
# ./catalina.sh restart
# 查看是否启动成功
cd /usr/local/soft/tomcat9
tail -f logs/catalina.out


```

### 配置tomcat打印字符编码

```
cd  ../tomcat9/conf
vi logging.properties
# 将conf/logging.properties文件中，下边一行的UTF-8改成GBK

java.util.logging.ConsoleHandler.encoding = UTF-8


```
